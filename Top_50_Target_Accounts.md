## How to Read the Columns

| Column | What it means |
|---|---|
| **MW / power** | Publicly discussed live capacity, contracted power, pipeline, or best available directional signal |
| **Site status** | `Live` · `Expansion` · `Under construction` · `Transition` · `Greenfield` · `Mixed` |
| **Power control layer** | See legend below |

### Power Control Layer Legend

| Label | Meaning |
|---|---|
| **Owns / controls power-heavy sites** | Strongest signal for direct infrastructure buying |
| **Utility-contracted** | Operator controls the campus but depends on grid / PPA / energy counterparties |
| **Leased / wholesale** | More dependent on third-party operators or contracted capacity |
| **Powered-land** | Land platform controls the site pipeline before the operator is chosen |
| **Hybrid** | Blends multiple models |
